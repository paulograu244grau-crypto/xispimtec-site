# Xispimtec - Site Profissional

Bem-vindo ao repositório do site da **Xispimtec** - Especialista em desenvolvimento de sites e soluções digitais para pequenas e médias empresas.

## 📋 Conteúdo

Este projeto inclui um site profissional one-page com as seguintes seções:

### Seções do Site

1. **Hero** - Apresentação impactante com CTA principal
2. **Sobre** - Apresentação da empresa, missão e diferenciais
3. **Serviços** - 6 serviços principais com benefícios
4. **Diferenciais** - 6 razões para escolher Xispimtec
5. **Portfólio** - 6 projetos fictícios com detalhes
6. **Depoimentos** - 3 depoimentos de clientes satisfeitos
7. **CTA Final** - Chamada para ação persuasiva
8. **Contato** - Formulário, informações de contato e WhatsApp
9. **Rodapé** - Links rápidos e informações adicionais

## 🎨 Design & Visual

- **Paleta de Cores**: Azul/Ciano (#2DD9D9) sobre fundo escuro
- **Tipografia**: Poppins (títulos) e Inter (corpo)

## 🔧 Como Usar

### Instalação
1. Baixe ou clone os arquivos
2. Abra `index.html` em um navegador
3. Ou hospede em um servidor web

### Configuração
Atualize os seguintes dados:

**Em `index.html`:**
- Número de WhatsApp (procure por `5547992505922`)
- E-mail de contato
- Links de redes sociais

**Em `script.js`:**
- Google Analytics ID (se usar tracking)
- Service Worker (se configurar PWA)

## 🎯 Otimizações Implementadas

### Performance
- Paleta de cores CSS usar variáveis (rápidas)
- Bundle CSS único (sem importações externas)
- JavaScript eficiente com lazy loading
- Animações CSS (mais eficientes que JS)
- Debounce e Throttle para eventos

### Conversão
- Multiple CTAs estrategicamente posicionados
- Formulário de contato simplificado
- WhatsApp como canal principal (urgência)
- Social proof com depoimentos
- Estatísticas impactantes

### UX/UI
- Navbar sticky com efeito visual
- Scroll suave
- Animações ao aparecer (fade-in)
- Feedback visual em cliques
- Indicadores de hover claros
